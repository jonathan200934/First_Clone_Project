About Github

To revert(go back) to the commit you want in your project(everything including files and changes! )


You can first take a look at your commit log at github or type in 'git log' in your terminal, I will use github as example

For example, I want to revert back to commit 123456, In my terminal I will type in 'git reset --hard (123456)'

If you wish your remote repository also rolled back to your local commit do: git push 'reponame' -f, notice it will erase all the commit after the commit you push!
